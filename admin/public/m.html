Lưu ý về tính năng Preview:
1. Tính năng này yêu cầu file .vtt (WebVTT).
2. File VTT này sẽ trỏ đến một tấm ảnh lớn (gọi là Sprite Sheet) chứa hàng loạt ảnh nhỏ ghép lại.
3. Ở ví dụ trên, tôi dùng file VTT demo nên hình ảnh hiện ra sẽ là phim hoạt hình (Sintel), không khớp với video Mux.
4. Để làm thật, bạn cần dùng FFmpeg để tạo file sprite và vtt từ video gốc của bạn.
